CL8

An entirely custom software suite for my 8-bit processor architecture, CL8.

Component Repositories

  • Hardware: a register-transfer-level implementation of the CL8 specification in SystemVerilog
  • Microcode: a C++-macro-engine-powered serialization program for my custom CL8 microcode
  • Emulator: a command line emulator for painless debugging of CL8 binaries
  • Assembler: a Flex/Bison-based tool that converts CL8 assembly to raw machine code
  • Disassembler: a binary analysis program that converts CL8 machine code back to human-readable source
  • Compiler: a custom compiler for my own programming language, Titanium, targeting the CL8 architecture

Titanium

A novel programming language compiler designed around the CL8 processor architecture, Titanium uses the GNU Flex/Bison lexer and parser generators to transform custom source code into flat CL8 assembly.

3-D Rendering

Voxelization

A modern-OpenGL-based voxel rendering program, geo utilizes geometry pre-generation, vertex batching, and indexed instancing for remarkable efficiency.

github.com/connorjlink/geo

Ray Tracing

By simulating light rays with physically based materials, raytracer produces photorealistic scene renders with reflections, soft shadows, and global illumination.

github.com/connorjlink/raytracer

Rasterization

Powered by a custom linear algebra library, rasterizer features frustum culling, depth buffering, and face-based shading to view arbitrary .obj models.

github.com/connorjlink/constexpr-linear-algebra-library
