What is haze?

Haze is a single-application software suite encompassing the entire from-the-ground-up toolchain supporting my own processor specification haze. Currently it includes an optimizing compiler for Titanium, my own programming language, targeting haze and a robust simulation framework to verify and validate the results of the compiler.
